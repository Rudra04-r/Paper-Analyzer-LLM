# Paper-Analyzer-LLM
# 🧠 AI-Powered Research Paper Assistant

An intelligent AI tool that automatically extracts, summarizes, and analyzes academic papers (PDFs). Built with Python, Streamlit, and Google's Gemini API to help researchers, students, and academics quickly grasp methodologies, results, and future work.

## 🌟 Features

* **📝 Automated Summarization:** Generates a concise, easy-to-understand summary of the entire paper.
* **🔍 Core Extraction:** Instantly isolates the **Problem Statement**, **Methodology**, and **Results**.
* **💡 Constructive Critique:** Suggests improvements and identifies potential flaws in the methodology.
* **🚀 Future Work:** Provides 3 actionable directions for future research based on the paper's findings.
* **📚 Recommendations:** Suggests similar foundational papers and concepts to explore next.

## ⚙️ Tech Stack

* **Language:** Python
* **UI Framework:** Streamlit
* **PDF Processing:** PyMuPDF (`fitz`)
* **LLM Engine:** Google Generative AI (Gemini 2.5 Flash)

## 🚀 How to Run in Google Colab (Easiest Method)

You can run this entire application directly in your browser using Google Colab without installing anything on your local machine.

1. Open a new [Google Colab Notebook](https://colab.research.google.com/).
2. Copy the full code block from `app.py` (provided in this repository) into a single Colab cell.
3. Run the cell. 
4. The notebook will print an **Endpoint IP**. Copy this IP address.
5. Click the generated `loca.lt` link, paste your Endpoint IP into the security check, and your app will launch!
